 HOF :-  higher order function

 Def:-    a function which accet another function as argment is called as HOF.
**************************************************************

callback function:-   a function passed as argument in another function.

**************************************************************


// forEach:

 The forEach is a higher order function which calls a function for every element in an array.
--------------------------------------------------------------------------------------------------------------------

// syntax:-   given arrayname.forEach(callback function(parameter){

                                                 statment
                                     }) 

-----------------------------------------------------------------------------------------------------------------------
 eg 1:-

let ages = [11,21,23,15,24,26,45,58,57,59]


ages.forEach(function(data){
    console.log(data)
})

o/p:-

11
21
23
15
24
26
45
58
57
59
---------------------------------------------------------------------------------------------------------------

To create a new array using forEach and push method.

eg:-


let ages = [11,21,23,15,24,26,45,58,57,59]

let loop = [];
ages.forEach(function(data)
{
    loop.push(data*2)
})


console.log(loop)

o/p:-

[22, 42, 46, 30, 48, 52, 90, 116, 114, 118]
